Detailed Description

Classe que deve armazenar dados de uma carona.

Esta classe e a representacao de uma carona no programa e possui como atributos um Código de Carona, uma Cidade de origem, um Estado de origem, uma Cidade de destino, um Estado de destino, uma Data, uma Duracao, a quantidade de Vagas e um Preco. A classe nao possui metodos de validacao pois assume-se que os dominios ja estarao criados (e portato validados)
