Detailed Description

Classe que deve armazenar dados de um usuario.

Esta classe e a representacao de um usuario no programa e possui como atributos um Nome, um Telefone, um Email, um CPF e uma Senha. A classe nao possui metodos de validacao pois assume-se que os dominios ja estarao criados (e portato validados)
