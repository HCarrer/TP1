\doxysubsection{Detailed Description}
Classe que deve armazenar dados de uma carona. 

Esta classe e a representacao de uma carona no programa e possui como atributos um Código de \mbox{\hyperlink{class_carona}{Carona}}, uma \mbox{\hyperlink{class_cidade}{Cidade}} de origem, um \mbox{\hyperlink{class_estado}{Estado}} de origem, uma \mbox{\hyperlink{class_cidade}{Cidade}} de destino, um \mbox{\hyperlink{class_estado}{Estado}} de destino, uma \mbox{\hyperlink{class_data}{Data}}, uma \mbox{\hyperlink{class_duracao}{Duracao}}, a quantidade de Vagas e um \mbox{\hyperlink{class_preco}{Preco}}. A classe nao possui metodos de validacao pois assume-\/se que os dominios ja estarao criados (e portato validados)
