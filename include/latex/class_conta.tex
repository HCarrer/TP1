\doxysubsection{Detailed Description}
Classe que deve armazenar dados de uma conta bancaria. 

Esta classe e a representacao de uma conta bancaria no programa e possui como atributos um código de Banco, um numero de Agencia e um numero de \mbox{\hyperlink{class_conta}{Conta}}. A classe nao possui metodos de validacao pois assume-\/se que os dominios ja estarao criados (e portato validados)
