\doxysubsection{Detailed Description}
Codigo destinado a identificar uma \mbox{\hyperlink{class_senha}{Senha}}. 

A validacao deste codigo apenas valida codigos no formato X\+X\+X\+XX onde X pode ser ser letra, digito, \#, \$, \% ou \&. Deve-\/se haver pelo menos uma letra e um digito. Nao se pode haver caracteres repetidos.
