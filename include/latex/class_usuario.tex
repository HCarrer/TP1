\doxysubsection{Detailed Description}
Classe que deve armazenar dados de um usuario. 

Esta classe e a representacao de um usuario no programa e possui como atributos um \mbox{\hyperlink{class_nome}{Nome}}, um \mbox{\hyperlink{class_telefone}{Telefone}}, um \mbox{\hyperlink{class_email}{Email}}, um \mbox{\hyperlink{class_c_p_f}{C\+PF}} e uma \mbox{\hyperlink{class_senha}{Senha}}. A classe nao possui metodos de validacao pois assume-\/se que os dominios ja estarao criados (e portato validados)
